# Coala Design - Portfolio

Este repositório contém o site oficial da **Coala Design**, disponível em [coaladsgn.com](https://coaladsgn.com).

## 🚀 Sobre o Projeto

O site [coaladsgn.com](https://coaladsgn.com) funciona como o **portfólio oficial** da Coala Design. Ele foi concebido para centralizar a exibição de projetos, demonstrar competências criativas e servir como o ponto de contato principal para clientes e parceiros.

O foco do projeto é a apresentação visual e a consolidação da identidade da marca no ambiente digital.
